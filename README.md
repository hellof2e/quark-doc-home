## 如何使用

```bash
npm install quark-doc-home
```


### 使用

无论是`Vue`，`React`，`Angular`还是`Jq`项目，该组件都可以被使用。

```js
// 引入
import 'quark-doc-home'

// 使用
<quark-doc-home />
```
